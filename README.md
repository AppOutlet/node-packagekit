# node-packagekit

node-packagekit is a library to integrate [packageKit](https://www.freedesktop.org/software/PackageKit/) with node applications.

## Installation

### Using NPM

```shell
$ npm install packagekit
```

### Using yarn

```shell
$ yarn add packagekit
```
